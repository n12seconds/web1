직접 웹서버 설치하기

아파치for윈도우 설치
binami WAMP stack(윈도우 아파치 mysql데이터베이스 php미들웨어)
이 매니저프로그램을 사용해서 끄고켜고 관리 가능

http://localhost/index.html
http://127.0.0.1/index.html
localhost와 같이 의미가 있는 이름 = 도메인 네임(domain name),
127.0.0.1 = ip 주소(ip address)
(localhost라는 도메인 이름은 내컴퓨터라는 의미)
위 주소들은 내컴퓨터의 index.html을 불러오라는 것.

index.html의 위치는 apache → htdocs → index.html
htdocs 디렉토리(hypertext documents 의 약자)를 비우고 내가 원하는 파일을 복붙하면 됨.
** 웹브라우저가 웹서버에 자료요청, 웹서버는 설정된 폴더에서 자료찾아서 웹브라우저로 전송, 웹브라우저는 그걸 해석해서 화면에 표시해주는것.

http:// 는 웹서버와 웹브라우저간의 통신을 통해 연 페이지.
(http = hyper text transfer protocol 웹페이지 전송규약)
file:// 은 브라우저가 파일을 직접 연 페이지.


그럼 다른 컴퓨터에서는 웹서버가 설치된 컴퓨터의 ip주소를 어떻게 알아낼수있을까?
두 컴퓨터가 같은 네트워크에 연결되어있어야함.
